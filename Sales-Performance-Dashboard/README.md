# Superstore Sales Performance Analysis Using Power BI
## Project Overview
This project analyzes sales performance, profitability, and product trends using Power BI. The objective was to transform raw sales data into actionable business insights through interactive visualizations and key performance indicators (KPIs). The dashboard enables stakeholders to monitor sales performance, identify profitable product categories, and evaluate regional sales trends.
